What is Playbook/Roles?
	It is a yaml file.
	Which Contains 
	- Multiple Play.

What is Play?
	Is a specification of 
	HOSTS: localhost or GROUP Name or GOG name
	TASK(S)
		- Collections of Modules with their own param.

	HOSTS: all
	Tasks:
		Task1 - Module1 - Param
		Task2 - Module2 - Param
		Task3 - Module3 - Param
		Task4 - Module4 - Param
	HOSTS: all
	Tasks:
		Task1 - Module - Param
		Task2 - Module - Param
		Task3 - Module - Param
		Task4 - Module - Param
	HOSTS: all
	Tasks:
		Task1 - Module - Param
		Task2 - Module - Param
		Task3 - Module - Param
		Task4 - Module - Param
